# Passart <sup>2</sup>: Secure passwords and sensitive documents in Art!

My algorithm encodes text input, such as passwords or other sensitive documents using a random encoding scheme each time the program is run. Each word is then mapped to an RGB color, thus allowing for the creation of a colorful patchwork of art.
